# Icon

`wc-icon` renders Tabler webfont icons with configurable size and color for inline iconography across components and layouts.

## Technical Details
### Element Contract
- Custom element tag: `wc-icon`
- Render model: Light DOM (Shadow DOM is disabled)
- Icon mechanism: renders an `<i>` element with classes in the format `ti ti-{name}`

### Properties
| Property | Type | Default | Allowed Values |
| --- | --- | --- | --- |
| `name` | string | empty string | Tabler icon names, for example `home`, `settings`, `heart` |
| `size` | number | `24` | Positive numeric pixel values |
| `color` | string | empty string | Any valid CSS color value |

### Behavior Notes
- The component returns no rendered output when `name` is empty.
- `size` is applied as inline `font-size` on the icon element.
- `color` is applied as inline `color`; empty value inherits surrounding text color.

### Usage Example
```html
<wc-icon name="home" size="20" color="#1d4ed8"></wc-icon>
```

### Accessibility Guidance
- The icon is decorative by default and rendered with `aria-hidden="true"`.
- If an icon needs semantic meaning, pair it with readable text outside this component.
