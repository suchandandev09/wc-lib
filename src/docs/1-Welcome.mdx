# Welcome to `wc-lib`! 🎉

Welcome to our globally-styled atomic Web Component library built with **Lit** and **Vite**.

This library differs from traditional web components by intentionally **disabling the Shadow DOM**. This allows our components to behave like standard Light DOM HTML elements, enabling you to style them seamlessly using overarching global SCSS.

## Core Features
- **Light DOM Architecture**: Components output raw, queryable HTML fragments without shadow root barriers.
- **Global SCSS Design Tokens**: Built on top of accessible CSS variables mapped globally.
- **Icon Webfont Integration**: Out-of-the-box support for over 5000+ icons via Tabler Icons.
- **Extensively Tested**: Every `<wc-*>` node behaves predictably through bundled Vitest integration testing.
